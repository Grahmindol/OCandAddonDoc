Description

This is the component provided by the energy pylon with the adapter.

Component name: draconic_rf_storage.

Methods

  • getEnergyStored():number Returns an integer value of the current energy stored in the multiblock.
  • getMaxEnergyStored():number Returns an integer value of the maximum energy that can be stored in the current tier.
  • getTransferPerTick():number Returns an integer corresponding to the RF gain or loss per tick.
  • canExtract():boolean Returns whether this component can have energy extracted from the connected side
  • canReceive():boolean Returns whether this component can receive energy from the connected side
