Description

This is the component provided by Draconic Reactor component with the adapter.

Component name: draconic_reactor.

Methods

  • stopReactor():boolean stop the Reactor.
  • canReceive():boolean Returns whether this component can receive energy from the connected side
  • chargeReactor():boolean charge the fuel in the reactor
  • canExtract():boolean Returns whether this component can have energy extracted from the connected side
  • activateReactor():boolean activate the reactor.
  • getEnergyStored():number Returns the amount of stored energy on the connected side.
  • getReactorInfo():table Returns an table with all information about the reactor. an example of table:
    {fuelConversionRate=0.0,energySaturation=0.0,fuelConversion=1.393,maxFieldStrength=100000000.0,maxEnergySaturation=1000000000.0,fieldStrength=50000000.0,temperature=2000.00,maxFuelConversion=10368.0,fieldDrainRate=1373.0,failSafe=false,generationRate=2.0,status="running"}
  • setFailSafe() set the reactor in safe mod.
  • getMaxEnergyStored():number Returns the maximum amount of stored energy on the connected side.
