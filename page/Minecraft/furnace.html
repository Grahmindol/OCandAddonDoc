Description

This is the component provided by the furnace block with the adapter.

Component name: furnace.

Methods

  • getBurnTime():number The number of ticks that the furnace will keep burning from the last consumed fuel.
  • getCurrentItemBurnTime():number The number of ticks that the currently burning fuel lasts in total.
  • getCookTime():number The number of ticks that the current item has been cooking for.
  • getTotalCookTime():number The number of ticks that the current item needs to cook.
  • isBurning():boolean Get whether the furnace is currently active.
