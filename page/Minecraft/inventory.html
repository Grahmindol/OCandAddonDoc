Description

This is the component provided by by all the blocks which contain item (and which are well coded) with the adapter.

must be activate in config file.

Component name: inventory.

Methods

  • getInventoryName():string Get the name of this inventory.
  • getInventorySize():number Get the number of slots in this inventory.
  • getSlotStackSize(slot:number):number Get the stack size of the item stack in the specified slot.
  • getSlotMaxStackSize(slot:number):number Get the maximum stack size of the item stack in the specified slot.
  • compareStacks(slotA:number, slotB:number):boolean Compare the two item stacks in the specified slots for equality.
  • transferStack(slotA:number, slotB:number[, count:number=math.huge]):boolean Move up to the specified number of items from the first specified slot to the second.
  • getStackInSlot(slot:number):table Get a description of the item stack in the specified slot.
  • getAllStacks():table Get a list of descriptions for all item stacks in this inventory.
