Description

This is the component provided by the note block with the adapter.

Component name: note_block.

Methods

  • getPitch():number Get the currently set pitch on this note block.
  • setPitch(value:number) Set the pitch for this note block. Must be in the interval [1, 25].
  • trigger([pitch:number]):boolean Triggers the note block if possible. Allows setting the pitch for to save a tick.
