Description

This is the component provided by the Alarm block.

Component name: os_alarm.

Methods

  • setAlarm(file_name:string) Allows you to change the alarm played, the default config is klaxon1 and klaxon2 and the default sound is klaxon1
  • setRange(range:number) Sets the range in blocks from the alarm block, allowed range is 0-15
  • activate() It.. activates the alarm.
  • listSounds():table returns a table of configured alarms.
  • deactivate() It.. deactivates the alarm.
